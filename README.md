2. Ajustar o cálculo do IMC

Fórmula:

IMC = peso / (altura * altura)

👉 Importante:

Altura geralmente vem em cm, então precisa converter para metros:
altura_m = altura / 100

3. Classificação do IMC

Use essa lógica:

IMC	Classificação
< 18.5	Magreza
18.5 - 24.9	Normal
25 - 29.9	Sobrepeso
30+	Obesidade

4. Arredondar para 2 casas decimais

Se estiver usando JavaScript:
